# astro-jyotish-website

एक आधुनिक ज्योतिष वेबसाइट जो Astro framework के साथ बनाई गई है।

## 🌟 विशेषताएं

- ⚡ Astro framework द्वारा तेज़ और optimized
- 🎨 सुंदर और प्रतिक्रियाशील डिज़ाइन
- 📱 मोबाइल-फर्स्ट approach
- 🔍 SEO optimized
- 🚀 Vercel पर deployed

## 🚀 शुरुआत करें

### आवश्यकताएं
- Node.js 16+ 
- npm या yarn

### इंस्टॉलेशन

```bash
# रेपोजिटरी को क्लोन करें
git clone https://github.com/akashvats189-droid/astro-jyotish-website.git
cd astro-jyotish-website

# निर्भरताएं इंस्टॉल करें
npm install

# डेवलपमेंट सर्वर शुरू करें
npm run dev
```

## 📁 प्रोजेक्ट संरचना

```
src/
├── components/     # Astro components
├── pages/          # Page routes
├── layouts/        # Layout templates
└── styles/         # CSS styles
```

## 🔧 विकास

```bash
# डेवलपमेंट मोड
npm run dev

# बिल्ड करें
npm run build

# प्रोडक्शन में preview करें
npm run preview
```
